# Antilities

## What is this?
Antilities is a pretty basic triggerbot mod which is meant to be hard to find if someone is doing a basic ss on you. It is designed to be lightweight and runs on Fabric 1.21.11.
### Servers That Detected It!
- mcpvp.club

## Features
* **Smart Delay:** Uses vanilla attack cooldowns to mimic natural player timing.
* **Target Delay:** Adds a customizable delay before attacking once a target is hovered.
* **Shield Awareness:** Option to ignore targets that are blocking with shields and pause while the user is shielding.
* **Filter Toggles:** Ability to toggle attacks on baby mobs.
* **Screen Safety:** Automatically pauses the triggerbot when any in-game menu or container is open.
* **Minimal Footprint:** Simple command-based configuration (`/-tbot`) and a single keybind toggle.
